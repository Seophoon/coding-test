# 石ゲーム 3\(9657番\)

### 問題

石ゲームは二人で楽しむ面白いゲームです。
テーブルの上に N 個の石があります。サンギュンとチャンヨンは交互にターンを取り、石を 1 個、3 個、または 4 個取ることができます。
最後に石を取った人がゲームに勝ちます。

二人が完璧にゲームを行った場合、勝つ人を求めるプログラムを作成してください。
     

### 入力

最初の行に N が与えられます。\(1 ≤ N ≤ 1000\)


### 出力

サンギュンがゲームに勝つ場合は "SK"、チャンヨンがゲームに勝つ場合は "CY" と出力してください。
